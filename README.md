# Installation de l'environnement python

## Prérequis

 - Python 3.5
 - [Pipenv](https://github.com/pypa/pipenv)

## Sous windows

- Installer l'environnement virtuel avec les dépendances

```bash
py -m pipenv install --dev --python=3.5
```

## Sous linux

- Installer l'environnement virtuel avec les dépendances

```bash
pipenv install --dev --python=3.5
```
